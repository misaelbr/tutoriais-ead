Arquivo

O recurso possibilita o compartilhamento de arquivos dentro da Sala Virtual.

Do ponto de vista pedagógico, como a ferramenta pode ser utilizada?

O recurso é útil para disponibilizar materiais de apoio, como slides, arquivos de textos nos mais variados formatos, apostilas, planilhas, imagens, entre outros.

Como todos os recursos disponíveis no Moodle, a ferramenta registra os acessos dos alunos ao material compartilhado, o que pode ser útil para o professor avaliar o engajamento dos alunos com o conteúdo.

Faça uso do recurso para disponibilizar templates de relatórios, artigos e trabalhos.

Quando o material é diponibilizado para leitura, dê preferência a formatos que possam ser utilizados sem a necesidade de instalação de softwares licenciados. Um bom exemplo para compartilhamento de textos é o formato PDF, pois o mesmo pode ser visualizado através do próprio Moodle, sem a necessidade de download.

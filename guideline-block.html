Estrutura visualmente a sala virtual

Do ponto de vista pedagógico, como pode ser utilizado?

É interessante para estruturar visualmente os blocos de conteúdos a partir de uma lógica visual.

Se utilizado com cuidado, ajuda a melhorar a aparência visual de uma sala virtual, além de facilitar a localização de conteúdos e atividades.

Verique os modelos disponíveis que disponibilizamos para uso.

Evite o uso de imagens excessivamentes grandes, pois isso afeterá negativamente o carregamento da página.

Não inclua informações importantes nas imagens, pois isso irá comprometer a acessibilidade no ambiente virtual.
